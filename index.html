Phenex

Phenex is an application for annotating character matrix files with ontology terms using the Entity-Quality syntax for describing phenotypes.

View the Project on GitHub phenoscape/Phenex

Running Phenex

From this page you can launch Phenex as a Java web start application. Click the orange button to the left to start.

Detailed instructions

Clicking the launch button will download a file named “phenex.jnlp”. If the Java web plugin is enabled in your browser, the JNLP file may run automatically. If not, locate the downloaded file and double-click it. You should then see a notice that Java is starting up:

Next the application files will be downloaded:

Once the application is ready, you will be warned that the application will have access to your computer (as any native program would). Check the box and press “Run”.
